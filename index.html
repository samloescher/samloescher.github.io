books

I do a lot of reading, and it largely falls into two categories; Christian and Programming. However I do sometimes venture outside into the world of fiction and enjoyed The Martian recently.

I started programming with Java which I learnt from O'Reilly's Head First Java. It was an excellent book which I would recommend to anyone picking up Java with little prior programming experience. It was through this book I picked up the foundations of Object Oriented design. The most influential programming book I have read since is Clean Code by Robert Cecil Martin. It covers a lot about a lot and sets the reader on an adventure of how to produce better and more maintainable code.

career aspirations

I enjoy working on cutting edge technology, tackling technically hard problems. Much of what I find enjoyable is in the processes rather than the end product. In as much, I find often I prefer to be making things faster and more elegant, than quickly cobbled together prototypes.

With a background in electronics and passion for software, I like to be working low down in the software stack.

hobbies

I enjoy watching programming videos and find myself watching videos outside my expertise so that I get a better appreciation as to why things are done the way they are. I particularly enjoy the content published by cppcon and ACCU, Kevlin Henney's talks on how write better, more maintainable code are favourites of mine. Mike Acton did a superb talk about Data Oriented design which changed how I view laying out memory, with particular reference to the cache.

To keep fit I enjoy bouldering and a bit of cycling. In my spare time I like to grow as many vegetables as I can, learn some new things at meetups, and maybe a bit more cycling!
